local-html-tools

ウェブブラウザで動く、ローカルHTMLツール集（Static Web App）

更新日: 2026-01-25

コンセプト

このツール群は、お使いのコンピュータ上で直接動作する、一連のシンプルなHTMLファイル（Static Web App）です。 ? 各ツール（HTMLファイル）をウェブブラウザで開くと、専用の入力画面が表示されます。そこにファイル名などの必要な情報を入力するだけで、FFmpegで使えるコマンドラインのテキストが自動的に作成されます。

主な特徴

  • ローカル動作: インターネット接続は不要で、お使いのPC内だけで完結します。
  • 簡単な操作: ウェブブラウザでファイルを開き、情報を入力するだけでコマンドが手に入ります。
  • 効率化: （FFmpeg系の場合）複雑なコマンドの構文を覚える必要がなく、手入力によるミスを防ぎます。

ツール一覧

🎬メディア処理（FFmpeg）

🧩FFmpeg ファイル結合

複数の.wavファイルを1つに結合するコマンドを生成します。

🔊FFmpeg 音量正規化 (Loudnorm)

.wavファイルの音量をラウドネス基準に合わせて正規化します。

✂️FFmpeg 切り出し (Trim)

音声/動画ファイルの一部を時間指定で切り出すコマンドを生成します。

🎧FFmpeg 音声変換

音声ファイルを指定の形式に変換するコマンドを生成します。

📺YouTube向け動画生成

一枚の画像と音声ファイルから、YouTube投稿用の動画ファイル(.mkv)を生成します。

🎵MP4からWAV抽出

mp4ファイルから音声をWAVとして取り出すコマンドを生成します。

🔁MP4音声差し替え（WAV）

動画の映像はそのままに、WAV音声へ差し替えるコマンドを生成します。

🔇無音区間検出

silencedetect で無音区間を検出するコマンドを生成します。

🔍検索

🔎find 検索

findでファイル/ディレクトリを検索するコマンドを生成します。

🌐Web/URL

🛒Amazon dp 抽出

Amazon URL から dp/ASIN を取り出します。

🔗Facebook識別子（fbclid）除去

URL から Facebook識別子（fbclid）を除去します。

🔤URL エンコード/デコード

URLのエンコードとデコードを行います。

📦MIME Base64

MIME Base64のエンコード/デコードを行います。

🧹UTM除去

URL から utm_* パラメータを削除します。

🔐パスワード

🛡️パスワードジェネレーター

選択した文字種と文字数でパスワードを生成します。
